# FMU-API

**FMU-API** — это бесплатная open-source реализация API Frontol Mark Unit для работы с маркированным товаром ("Честный ЗНАК").

## Ключевые возможности

- Поддержка всех версий Frontol начиная с 6.20.5
- Интеграция с ТС ПИоТ
- Интеграция с локальным модулем честного знака
- Интеграция с TrueApi Честного знака (подключение ЭЦП)
- Работа с базой данных (CouchDb) и без
- Центральный модуль для мониторинга и обновлений
- Простая установка на Windows, Docker и Linux
